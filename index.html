How to Use

  1. Use the Brush Mode dropdown to select what to draw.
  2. Click and drag on the grid to create walls, goals, and traps.
  3. Click Start Training to watch the agent learn.
  4. The grid cells will colorize to show the agent's "knowledge": green is good, red is bad.
  5. Click Stop Training to pause and Reset World to start over.
